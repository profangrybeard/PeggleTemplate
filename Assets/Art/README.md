# Art

All visual assets for the game. Organized into subfolders by purpose.

## Subfolders

### Sprites/
Individual game object images:
- `ball.png` — The player's ball
- `peg_blue.png` — Standard peg sprite
- `peg_orange.png` — Target peg sprite
- `peg_green.png` — Special peg sprite
- `peg_purple.png` — Bonus peg sprite
- `peg_glow.png` — Glow effect for hit pegs
- `launcher.png` — The aiming cannon
- `bucket.png` — Free ball bucket

### Backgrounds/
Full-screen background images for levels.

### UI/
User interface elements:
- Score display frames
- Ball counter icons
- Button sprites
- Panel backgrounds

## Import Settings

For pixel-perfect 2D sprites in Unity:
- Texture Type: Sprite (2D and UI)
- Pixels Per Unit: Match your design (commonly 100)
- Filter Mode: Point (for pixel art) or Bilinear (for smooth art)
